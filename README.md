# ubupy

(Ubu)ntu based Docker image with (Py)thon.

## Info

* Based on Ubuntu LTS (18.04)
* Python 2.7 and 3 + pip
* [confd](https://github.com/kelseyhightower/confd) onboard
* other tools:

  * curl
  * wget
  * git
  * gcc
  * tree
